# BookTrack Management System

BookTrack is a web-based reader community platform developed as a group
project for CSC575: Software Engineering – Theories and Principles.

The platform allows readers to discover books, organise their reading
lists, share reviews, and engage with other readers without involving
book purchasing or other e-commerce activities.

## Project Objectives

- Provide a central platform for readers to discover and discuss books.
- Allow users to organise books based on their reading status.
- Enable users to publish reviews that may help other readers.
- Provide administrators with book, user, and review moderation functions.
- Apply software engineering principles through requirements analysis,
  system design, implementation, and testing documentation.

## Main Features

### Reader

- Register and manage a user profile
- Browse and search for books
- View book descriptions
- Add books to reading lists
- Record books as read or planned for reading
- Submit book reviews
- Read reviews submitted by other users

### Administrator

- Manage books
- Moderate submitted books
- Moderate users
- Manage reviews
- View administrative records

## Software Engineering Documentation

- Software Requirements Analysis (SRA)
- Software Design Document (SDD)
- Software Test Document (STD)

## Technologies Used

- Java
- JavaServer Pages (JSP)
- HTML

- JavaScript
- MySQL

## Project Scope

BookTrack focuses on reader interaction and personal reading management.
It does not provide book purchasing, payment, delivery, or other
e-commerce services.

## Project Status

Academic prototype developed for CSC575. Some functions may use
prototype data and may not represent a production-ready application.
